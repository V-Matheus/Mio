# Diretrizes Gerais do Projeto

## Stack Tecnológica
- **React 19**
- **TypeScript** (modo strict)
- **Tailwind CSS v4**
- **Biome** para Lint e Formatação
- **TypeScript** para typecheck
- **Turborepo** para monorepo

## Regras de Execução e Desenvolvimento
- **Formatação de Código**: O código do projeto deve sempre seguir os padrões e regras do Biome.
- **Segurança**: Respeite estritamente as permissões definidas em `.claude/settings.json`.
- **Confirmação Obrigatória**: Qualquer modificação nos arquivos de código deve ser previamente confirmada com o usuário antes de ser persistida.
- **Padrões de UI**: Para criação de componentes baseados em design, siga o guia em `.claude/agents/design-to-react.md`.
